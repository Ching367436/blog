題解 (Stage 2)

這題要讓 Number(input)==1024 且 safeEval(input)==532。可以看出 0o1024 == 532，所以應該跟進位轉換有關。翻到了下面這個文件：

  • A leading 0 digit does not cause the number to become an octal literal (or get rejected in strict mode).

    https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number#number_coercion

所以用 01024 就能解了。原理是 Number(01024) 根據上面的文件，會被視為十進位的數字；而 01024 直接 eval 則是會是八進位的。
